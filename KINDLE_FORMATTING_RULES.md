# KINDLE FORMATTING RULES

## Structure
- Title page
- Copyright
- Dedication
- Table of Contents
- Chapters start on new page
- No extra spaces

## Text
- 1.2–1.4 line spacing
- Standard serif font
- No indentation on first paragraph of chapter
- Indentation on all other paragraphs

## Scenes
- Use *** or — • — for scene breaks

## Metadata
- Keywords for psychological thriller
- BISAC: FIC022040 (Thrillers), FIC025000 (Psychological)

## Export
- EPUB 3
- XHTML clean formatting

Output must be immediately usable in Kindle Direct Publishing.
